🚗 Road Accident Analysis Dashboard
📌 Project Overview

This project presents an interactive Road Accident Analysis Dashboard built using Microsoft Excel. The dashboard provides insights into accident casualties based on severity, vehicle type, road type, road surface, light conditions, area, weather conditions, and yearly trends.

🎯 Objectives
Analyze road accident casualty data.
Identify accident severity patterns.
Compare yearly casualty trends.
Understand the impact of road conditions and vehicle types on accidents.
Support data-driven decision-making for road safety improvements.
📊 Dashboard Features
Total Casualties Overview
Fatal, Serious, and Slight Severity Analysis
Vehicle-wise Casualty Distribution
Monthly Casualty Trend Comparison
Road Type Analysis
Road Surface Condition Analysis
Light Condition Analysis
Urban vs Rural Casualty Analysis
Interactive Filters (Year, Week Status, Weather Conditions)
🛠️Tools Used
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers
Dashboard Design & Data Visualization
📈 Key Insights
Majority of casualties occurred on Single Carriageways.
Most accidents happened during Daylight conditions.
Urban areas recorded higher casualties compared to rural areas.
Cars accounted for the highest number of casualties.
